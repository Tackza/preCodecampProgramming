p align จัดวางตำแหน่ง
div align จัดวางตำแหน่ง
img align ระบุการจัดตำแหน่งของภาพให้เป็นไปตามองค์ประกอบโดยรอบ alt text ระบุข้อความสำรองสำหรับภาพ border ระบุความกว้างของเส้นขอบรอบภาพ crossorigin anonymous use-credentials อนุญาตให้ภาพจากเว็บไซต์ของบุคคลที่สามที่อนุญาตให้เข้าถึงข้ามจุดที่จะใช้กับผืนผ้าใบ height ระบุความสูงของภาพ hspace ระบุช่องว่างด้านซ้ายและด้านขวาของภาพ ismap ระบุภาพเป็นฝั่งเซิร์ฟเวอร์ภาพแมป longdesc ระบุ URL ไปยังรายละเอียดของภาพ src ระบุ URL ของภาพ usemap ระบุภาพเป็นฝั่งไคลเอ็นต์ภาพแมป vspace ระบุช่องว่างด้านบนและด้านล่างของภาพ width ระบุความกว้างของภาพ
a link กำหนดตำแหน่งที่ตั้งที่จะไป
ul compact ระบุว่ารายการควรจะทำให้มีขนาดเล็กกว่าปกติ
type ระบุชนิดของเครื่องหมายที่จะใช้ในรายการ
ol compact ระบุว่ารายการควรจะทำให้มีขนาดเล็กกว่าปกติ
reversed ระบุว่าการสั่งซื้อรายการที่ควรจะลงมา (9,8,7...)
start ระบุค่าเริ่มต้นของรายการสั่งซื้อ
type ระบุชนิดของเครื่องหมายที่จะใช้ในรายการ
table align ระบุการจัดตำแหน่งของตารางตามข้อความรอบ
bgcolor ระบุสีพื้นหลังของตาราง
border ระบุหรือไม่ว่าตารางจะถูกใช้เพื่อวัตถุประสงค์ในการจัดวาง
cellpadding ระบุช่องว่างระหว่างผนังเซลล์และเนื้อหาเซลล์
cellspacing ระบุช่องว่างระหว่างเซลล์
frame ระบุว่าส่วนของเส้นขอบนอกที่ควรจะมองเห็น
rules ระบุส่วนของพรมแดนภายในที่ควรจะมองเห็น
sortable ระบุว่าตารางที่ควรจะจัดเรียง
summary ระบุสรุปเนื้อหาของตารางที่
width ระบุความกว้างของตาราง
form accept ระบุรายการคั่นด้วยเครื่องหมายจุลภาคประเภทไฟล์ที่เซิร์ฟเวอร์รับ (ที่สามารถส่งผ่านการอัปโหลดไฟล์)
accept-charset กำหนดการเข้ารหัสตัวอักษรที่จะใช้สำหรับการส่งแบบฟอร์ม
action ระบุว่าจะส่งแบบฟอร์มข้อมูลเมื่อส่งแบบฟอร์ม
autocomplete ระบุว่ารูปแบบที่ควรจะมีการเติมข้อความอัตโนมัติหรือปิด
enctype ระบุว่าข้อมูลรูปแบบที่ควรจะเข้ารหัสเมื่อส่งไปยังเซิร์ฟเวอร์ (เฉพาะ method="post" )
method ระบุวิธีการ HTTP ที่จะใช้เมื่อการส่งแบบฟอร์มข้อมูล
name ระบุชื่อของฟอร์ม
novalidate ระบุว่ารูปแบบที่ไม่ควรได้รับการตรวจสอบเมื่อส่ง
target ระบุตำแหน่งที่จะแสดงการตอบสนองที่ได้รับหลังการส่งแบบฟอร์ม
button autofocus ระบุว่าปุ่มควรมุ่งเน้นโดยอัตโนมัติได้รับเมื่อโหลดหน้าเว็บ
disabled ระบุว่าปุ่มควรจะปิดการใช้งาน
form ระบุหนึ่งหรือมากกว่าหนึ่งรูปแบบปุ่มเป็น
formaction ระบุว่าจะส่งแบบฟอร์มข้อมูลเมื่อส่งแบบฟอร์ม เฉพาะ type="submit"
formenctype ระบุว่าข้อมูลรูปแบบควรมีการเข้ารหัสก่อนที่จะส่งไปยังเซิร์ฟเวอร์ เฉพาะ type="submit"
formmethod ระบุวิธีการส่งรูปแบบข้อมูล (ซึ่งวิธีการที่จะใช้ HTTP) เฉพาะ type="submit"
formnovalidate ระบุว่ารูปแบบข้อมูลที่ไม่ควรได้รับการตรวจสอบการส่ง เฉพาะ type="submit"
formtarget ระบุตำแหน่งที่จะแสดงการตอบสนองหลังจากส่งแบบฟอร์ม เฉพาะ type="submit"
name ระบุชื่อสำหรับปุ่ม
type ระบุชนิดของปุ่ม
value ระบุค่าเริ่มต้นสำหรับปุ่ม
